PROCESSING VERSION - 4.2

LIBRARIES REQUIRED:
Processing's sound library (processing.sound.* [Sound 2.3.1])
Java's robot class (java.awt.*)

CONTROLS:
AWSD - Move
SPACE - Jump
LEFT_MOUSE_BUTTON - Primary fire
RIGHT_MOUSE_BUTTON - Alternate fire
R - Play/Restart
ESC - Quit
ARROW_KEYS - Adjust mouse sensitivity

SAVE FILE:
A default save file is located in the data directory with default values. If you want reset your
save, you can just copy/paste the file into the save file the game loads and writes to.
